# Graph Traversal and Representation System

## Project Overview

This project demonstrates graph representation using adjacency lists.
It includes BFS and DFS traversal algorithms.

## Classes

### Vertex
Represents a graph node.

### Edge
Represents a connection between vertices.

### Graph
Stores the graph using adjacency lists.

### Experiment
Runs traversal tests and measures execution time.

## BFS

Breadth-First Search visits nodes level by level.

Time Complexity:
O(V + E)

## DFS

Depth-First Search explores deeply before backtracking.

Time Complexity:
O(V + E)
